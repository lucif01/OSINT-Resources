# Metadata extraction

> Last updated: 2026-07-23 · 5 curated resources

## Introduction

File metadata can provide provenance clues, software history, and timestamps. Work on copies, preserve original hashes, and remember that metadata can be stripped or falsified.

## Resources

| Resource | Official URL | Description | Platform | Access | Open source |
|---|---|---|---|---|---|
| MediaInfo | [mediaarea.net](https://mediaarea.net/en/MediaInfo) | Displays technical and container metadata for audio and video files. | Linux/macOS/Windows | Free | Yes |

## Notes

Do not alter originals. Keep hash, source, collection date, and any transformation steps in the case record.
