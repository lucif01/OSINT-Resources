# Code hosting intelligence

> Last updated: 2026-07-23 · 7 curated resources

## Introduction

Public source-code hosts can reveal project history, security advisories, packages, and official technical documentation. Respect licenses, access controls, and researcher scope; never use this section to target private repositories or exposed credentials.

## Resources

| Resource | Official URL | Description | Platform | Access | Open source |
|---|---|---|---|---|---|
| GitHub code search | [github.com](https://github.com/search) | Official search interface for public code, issues, discussions, and repositories. | Web | Free/Paid | No |
| GitHub Advisory Database | [github.com](https://github.com/advisories) | Maintained vulnerability advisories with ecosystem and dependency context. | Web/API | Free | No |
| GitHub Archive | [www.gharchive.org](https://www.gharchive.org/) | Public archive of GitHub event data for historical research and reproducible analysis. | Web/Data | Free | No |
| GitLab search | [gitlab.com](https://gitlab.com/explore) | Official discovery interface for public GitLab projects and groups. | Web | Free/Paid | No |
| Gitleaks | [github.com](https://github.com/gitleaks/gitleaks) | Open-source secret scanner for repositories you own or are authorized to audit. | Linux/macOS/Windows | Free | Yes |
| TruffleHog | [github.com](https://github.com/trufflesecurity/trufflehog) | Verifies potential credentials in source and storage sources during authorized security reviews. | Linux/macOS/Windows | Free | Yes |
| OSV | [osv.dev](https://osv.dev/) | Open vulnerability database with package-version search and API access. | Web/API | Free | Yes |

## Notes

Use results to improve security, attribution, and evidence quality. Responsible disclosure applies when public material exposes a genuine security issue.
