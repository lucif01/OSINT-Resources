## Featured tools

- Amass for asset discovery.
- MISP for structured threat-intelligence sharing.
- SpiderFoot for repeatable OSINT collection.
- ExifTool for media metadata inspection.
- OpenCorporates for company-register research.

## Beginner guide

1. Define a lawful research question and document scope.
2. Start with primary records and corroborate significant claims.
3. Preserve URLs, timestamps, captures, and methods.
4. Minimize personal-data collection and do not bypass access controls.
