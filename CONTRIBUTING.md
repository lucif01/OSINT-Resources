# Contributing

Thanks for improving this directory. Submit only resources that are lawful to use, publicly documented, and useful to an investigator.

## Entry standard

Each entry must include an official URL, an original one-sentence description, platform, access level, and open-source status. Check for existing links first. Do not add credential-harvesting, doxxing, access-bypass, or unverified resources.

## Pull requests

Keep one topic per pull request, run the validation scripts, and explain why the resource is maintained and distinct. Report broken links or changed pricing through an issue.
